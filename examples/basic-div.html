Basic Demo - Div

This is a basic demo of jQuery Backstretch being applied to a div with a class of 'container' rather than the body. You can of course change this to be almost any block level element and it also handles multiple instances.


<script src="../lib/jquery-1.6.4.min.js"></script>
<script src="../jquery.backstretch.min.js"></script>
<script>
    $('.container').backstretch("pot-holder.jpg");
</script>
